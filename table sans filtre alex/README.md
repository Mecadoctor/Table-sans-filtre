# La Table Sans Filtre

Site web pour le podcast vidéo "La Table Sans Filtre" animé par Alex Rizk.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS

## Installation

```bash
npm install
```

## Configuration

Créez un fichier `.env.local` à la racine avec :

```
ACCESS_CODES=code1,code2,code3
NEXT_PUBLIC_CALENDLY_URL=https://calendly.com/votre-lien
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```
